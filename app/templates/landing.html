Hvordan det fungerer

Fra fil til kunnskap, automatisk

Hvert dokument går gjennom en pipeline som leser ut tekst og struktur, deler det opp i meningsfulle biter og lager matematiske representasjoner som lar AI søke på betydning.

Filer hentes inkrementelt fra OneDrive, parses lokalt med Unstructured, embeddes med OpenAI og/eller Gemini, og lagres i to separate Pinecone- indekser. Eksterne LLM-er kobles på via MCP.

  1. 1

    Overvåking

    Vi spør Microsoft Graph hvert n-te minutt om hva som har endret seg i alle OneDrive- og SharePoint-områder. Bare nye eller oppdaterte filer trekkes ned — ingen full re-indeksering hver runde.

  2. 2

    Parsing med Unstructured

    Filer leses med Unstructured — PDF-er får layout-deteksjon og OCR (norsk + engelsk), Office-dokumenter parses direkte. Resultatet er ren tekst og metadata, oppdelt i naturlige seksjoner.

  3. 3

    Embedding

    Hver tekstbit sendes til OpenAI text-embedding-3-large som returnerer en 3072-dimensjonal vektor. Disse vektorene fanger betydning, ikke bare ord.

  4. 4

    Lagring i Pinecone

    Vektorene skrives til Pinecone sammen med metadata (filnavn, sti, eier, sist endret). Når en fil oppdateres, slettes de gamle vektorene før de nye legges inn — så indeksen er alltid synkron med OneDrive.

Funksjoner

Bygget for tenant-skala drift

Inkrementell delta-sync

Bruker Microsoft Graphs delta-API per drive. Etter første kjøring henter vi bare endringer — ekstremt billig og raskt.

Alle filtyper — også DWG

PDF, Word, PowerPoint, Excel, bilder, e-post, HTML/Markdown og over 25 andre formater. AutoCAD DWG og DXF indekseres med titteltabell, lagnavn, blokker og alle tekst-annotasjoner — pluss visuell PNG-render for multimodal søk.

Per-drive isolasjon

Hver brukers OneDrive får sitt eget Pinecone-namespace, så søk kan filtreres til spesifikke personer eller team.

Selvtilhelende

Eksponensielle retries på Graph- og Pinecone-feil, automatisk re-sync når Graph inkrementelle tokens utløper.

Norsk OCR

Tesseract konfigurert med norsk og engelsk språkpakke — skannede PDF-er får tekst.

App-only auth

Microsoft Entra ID client credentials. Ingen brukerkontekst, ingen samtykke å fornye — tjenesten kjører selvstendig.

Bygget for byggebransjen

CAD-tegninger, fotografier og dokumenter — alt i samme søk

Byggkons faglige hverdag handler ikke bare om Word-dokumenter. {{ brand_name }} møter dere der dere er — tekniske tegninger, fotografier fra befaring, teknisk korrespondanse og kontrakter.

AutoCAD DWG & DXF

DWG konverteres til DXF med libredwg, parses med ezdxf, og gir deg titteltabell, prosjekt-metadata, lagnavn, blokk-symbol, dimensjoner og alle tekst-annotasjoner som søkbart innhold. Tegningen rendres også til PNG som Gemini kan se direkte — så du kan finne tegninger basert på visuelt likhet, ikke bare ord.

To AI-modeller parallelt

Velg mellom OpenAI (GPT-5-serien) og Google Gemini 3 (multimodal — kan se bilder og tegninger), eller kjør begge samtidig i to indekser. To chat-paneler side om side i admin lar dere sammenligne svarene direkte.

Koble på alle LLM-er via MCP
